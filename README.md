# 流萤主题机械表盘

## 简介
FireflyWatchFace-mechanical 是一个概念性手表表盘项目，旨在把机械表盘设计与“流萤”主题的氛围结合起来。

> 说明：虽然现在的外观看起来可能没那么像“流萤”主题，但我会尽力把主题感表现出来。

## 兼容性
- 适配：Wear OS 3+（Android 11 及以上）
- 未测试：Android 9 未经测试，可能存在兼容性问题

## 特性
- 机械指针风格的表盘设计
- 简洁、低功耗的概念实现（无背景以降低资源占用）

## 注意事项
- 这是一个概念表盘：优先考虑简洁界面与低能耗，故很可能不会为当前版本添加动态/完整背景。

- 根据使用反馈逐步优化主题表现和兼容性
